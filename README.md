# Databricks-14-Days-AI-Challenge

This repository documents my learning journey through the **Databricks 14 Days AI Challenge** organized by **Indian Data Club**.

The goal of this repository is to:
- Capture daily learnings in a structured and reusable format
- Document hands-on experiments performed on **Databricks Free Edition**
- Note practical challenges, limitations, and workarounds encountered during the learning process
- Build a reference for Databricks fundamentals, SQL, Spark, and AI-related concepts
